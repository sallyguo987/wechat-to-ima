# 🪑 wechat-to-ima

> 微信公众号文章搜索 → 智能去重筛选 → 一键导入IMA知识库

**wechat-to-ima** 是一个 WorkBuddy 技能包，用于批量搜索微信公众号文章，经过去重和日期过滤后，通过 IMA OpenAPI 自动导入到指定知识库中。适用于舆情监测、行业研究、竞品追踪等场景。

---

## ✨ 功能演示

### 1. 多关键词搜索与智能去重

```
📝 搜索关键词:
   南康家具、南康 实木家具、赣州南康家具、
   南康家居产业、南康家具产业

📊 搜索结果:
   原始文章: 220 篇
   去重后:   152 篇
    
📅 日期过滤 (2026-04-01 ~ 2026-06-23):
   范围内:   10 篇 ✅
   范围外:   142 篇
```

### 2. HTML 浏览页面自动生成

搜索完成后自动生成自包含的 HTML 页面，支持：
- 🔍 标题/摘要关键词搜索
- 📅 按日期范围筛选
- 📋 一键复制文章链接
- 📤 导出为 JSON/CSV

### 3. IMA 知识库批量导入

```python
# 运行导入脚本
python scripts/ima_import.py \
  --client-id "your_client_id" \
  --api-key "your_api_key" \
  --kb-id "your_kb_id" \
  --input filtered.json
```

```
✅ 南康家具产业高质量发展论坛举办          → 已导入
✅ 南康实木家具出口增长30%                 → 已导入
✅ 赣州南康家具博览会即将开幕              → 已导入
... (共 10 篇)
📊 导入完成: 10/10 成功
```

## 🚀 功能特性

| 功能 | 说明 |
|------|------|
| 🔎 多关键词搜索 | 支持多个关键词并发搜索，覆盖不同角度 |
| 🧹 智能去重 | 基于文章ID自动去重，避免重复导入 |
| 📅 日期过滤 | 自定义时间范围，精准筛选目标文章 |
| 🌐 HTML浏览 | 生成自包含的搜索/筛选/导出页面 |
| ☁️ IMA导入 | 通过IMA OpenAPI直接导入知识库 |
| 🔄 URL解析 | 搜狗搜索结果URL → 微信原文URL解析 |

## 🔧 快速上手

### 前置条件

- WorkBuddy 客户端
- IMA 知识库（需获取 Client ID 和 API Key）

### 安装

1. 下载本仓库
2. 将 `wechat-to-ima` 文件夹放入 WorkBuddy 技能目录
3. 配置 IMA 凭证

### 使用

在 WorkBuddy 对话中通过以下关键词触发：

```
搜索微信公众号文章 + 南康家具 + 2026年4月至6月
导入IMA知识库
微信文章批量导入
```

## 📊 工作流程

```
┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│ 搜狗微信搜索  │ →  │  去重+过滤   │ →  │  生成HTML    │
│ 多关键词并发  │    │  日期范围    │    │  浏览页面    │

                                               ↓
┌──────────────┐    ┌──────────────┐    ┌──────────────┐

└──────────────┘    └──────────────┘    └──────────────┘
```

---

## ⚠️ 注意事项

- 搜狗微信搜索存在反爬机制，URL解析可能需要替代方案（如搜索引擎查找新闻转载链接）
- IMA OpenAPI 的 delete_knowledge 接口暂不可用，无效条目需在IMA客户端手动清理
- 搜索结果上限为每个关键词 50 篇，超出部分不会被捕获
